# URA Project – UI/UX & Color Theory Research

This project documents my undergraduate research work as a URA with a professor Oliver Schneider and TA Punit Kunjam, focused on understanding how color theory and perception influence UI/UX design decisions.

## Overview

The goal of this research was to move beyond subjective design choices and study how structured principles of color theory can improve usability, accessibility, and visual hierarchy in digital interfaces.

We explored how designers use color systems to:
- Improve readability and contrast
- Create consistent visual hierarchies
- Guide user attention through interfaces
- Support accessibility for different vision conditions
- Evoke emotional and cognitive responses through color

## Key Areas of Study

### 1. Color Theory in UI Design
Studied how traditional color theory (hue, saturation, brightness) translates into modern interface design, and how designers use structured palettes instead of ad-hoc color selection.

### 2. Visual Hierarchy & Readability
Explored how contrast and color relationships affect scanning behavior, attention flow, and content comprehension in UI layouts.

### 3. Accessibility & Inclusive Design
Reviewed how color choices impact accessibility, including readability for users with color vision deficiencies and low-contrast sensitivity.

### 4. Practical UI Experiments
Applied research insights to small UI prototypes and design exercises, testing how different color systems influence clarity and usability.

## Outcomes

- Improved understanding of systematic UI design principles
- Stronger ability to evaluate interfaces beyond personal preference
- Experience connecting design theory with practical application
- Better intuition for building accessible and consistent UI systems

## Repo Purpose

This repository serves as a structured record of research notes, experiments, and applied learning from the URA project in UI/UX and color theory.
